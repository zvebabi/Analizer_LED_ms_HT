8 // NOTE: 2011-01-07 Gill Ross
9 // Rob Tillaart's Statistic library uses one-pass of the data (allowing
10 // each value to be discarded), but expands the Sum of Squares Differences to
11 // difference the Sum of Squares and the Average Squared. This is susceptible
12 // to bit length precision errors with the float type (only 5 or 6 digits
13 // absolute precision) so for long runs and high ratios of
14 // the average value to standard deviation the estimate of the
15 // standard error (deviation) becomes the difference of two large
16 // numbers and will tend to zero.
17 //
18 // For small numbers of iterations and small Average/SE th original code is
19 // likely to work fine.
20 // It should also be recognised that for very large samples, questions
21 // of stability of the sample assume greater importance than the
22 // correctness of the asymptotic estimators.
23 //
24 // This recursive algorithm, which takes slightly more computation per
25 // iteration is numerically stable.
26 // It updates the number, mean, max, min and SumOfSquaresDiff each step to
27 // deliver max min average, population standard error (standard deviation) and
28 // unbiassed SE.

72 // adds a new value to the data-set
73 void Statistic::add(double value)
74 {
75  if (_cnt == 0)
76  {
77  _min = value;
78  _max = value;
79  } else {
80  if (value < _min) _min = value;
81  else if (value > _max) _max = value;
82  }
83  _sum += value;
84  _cnt++;
85 
86 #ifdef STAT_USE_STDEV
87  if (_cnt > 1)
88  {
89  _store = (_sum / _cnt - value);
90  _ssqdif = _ssqdif + _cnt * _store * _store / (_cnt-1);
91  }
92 #endif
93 }

97 {
